조건문 : if / else if / else 등의 문장을 사용해서 조건에 따라 다른 코드 블록 실행.
      
       1. if : 주어진 조건을 평가하고 참일 경우 특정 코드 실행.
          *
          if(조건문) { 참이면 실행
	      } 거짓이면 종료
	  
	   2. if-else : 프로그램 조건에 따라 두 가지 서로 다른 코드 블록 중 하나를 실행하는 제어구조.
		            만약 if문의 조건이 참이면 if 내에 있는 블록이 실행되고 조건이 거짓이면 else 블록이 실행된다.
		            이를 통해 두 가지 상황에 따라 다른 동작을 수행할 수 있다.
		            *
		            if (조건) { 조건이 참일때 실행되는 코드
		            } 조건이 거짓일때 실행되는 코드
	 
	   3. if-else if-else : 여러개의 조건을 연속적으로 평가. 그 중 첫번째 참인 조건에 해당하는 코드 블록을 실행하는 구조.
		                    만약 if나 else if의 조건도 참이 아니라면 마지막 else 블록 실행. 여러개의 조건을 추가할 수 있다.
		                    *
		                    if(조건1){
		                    조건1이 참일 때 실행되는 코드
		                    } else if(조건2){
		                    조건2가 참일 때 실행되는 코드
		                    } else{
		                    위의 모든 조건이 거짓일 때 실행되는 코드
		                    }
