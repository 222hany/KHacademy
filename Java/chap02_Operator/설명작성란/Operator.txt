연산자(Operator) : 연산에 사용하는 기호.

1. 산술연산자 (+, -, *, /, %) : 숫자 데이터 타입의 변수들을 다룰 때 사용. 기본적인 산술 연산을 실행.
2. 대입연산자 (=) : 변수에 값을 할당할 때 사용. 오른쪽 피연산자의 값을 왼쪽 피연산자에 할당.
3. 증감연산자 : 변수의 값을 1씩 증가시키거나 감소시킬때 사용.
           1)전위 증감 연산자(++x, --x) : ++ 항의 값에 1을 더한다.
           2)후위 증감 연산자(x++, x--) : -- 항의 값에 1을 뻰다.
4. 비교연산자 (<, >, ==) : 값을 비교할 때 사용. 크기나 등호비교 등을 수행할 수 있음. 두 개의 항 중 어느것이 큰지 작은지,
                        같은지 다른지의 여부를 판단. 비교연산자의 결과 값은 참 또는 거짓으로 반환.
                        > 왼쪽 항이 크면 참, 아니면 거짓을 반환.
		                < 왼쪽 항이 작으면 참, 아니면 거짓을 반환.
		                >= 왼쪽 항이 오른쪽 항보다 크거나 같으면 참, 아니면 거짓을 반환.
		                <= 왼쪽 항이 오른쪽 항보다 작거나 같으면 참, 아니면 거짓을 반홤.
		                == 두 항의 값이 같으면 참, 아니면 거짓을 반환.
		                != 두 항의 값이 다르면 참, 아니면 거짓을 반환.
