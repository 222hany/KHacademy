package : 어떠한 폴더 위치를 작성하는 공간. 내가 만들고 싶은 파일 경로(폴더위치)를 설정할 수 있음.

자바 명명규칙 : 클래스나 패키지 변수 상수 메소드 등 자바 프로그램을 할 때 이름을 따라할 규칙.
1. 카멜케이스 : 이름에 여러 단어를 혼합해서 사용한다면 첫번때 단어는 소문자, 두번째 단어부터는 대문자로 시작. 이름을 지을때 숫자를 먼저 입력해도 안됨.
            ex) firstName / lastName / phoneNumber
2. 스네이크케이스 : 모든 문자의 첫자는 소문자로 표기하고 단어 사이에는 언더바(_)를 사용.
               ex) first_name / last_name / phone_number
3. 파스칼케이스 : 모든 문자의 첫자를 대문자로 표기. 중간에 대문자를 넣어 표기 가능.
              ex) FirstName / LastName / PhoneNumber
4. 케밥케이스 : 모든 문자의 첫자는 소문자로 표기하고 문자들 사이에는 하이픈(-)연결.
             ex) first-number / last-number / phone-number
